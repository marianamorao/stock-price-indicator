## Project Motivation

Investment firms, hedge funds and individual traders use financial models to understand market behaviour and make profitable decisions. This project builds a **stock price predictor** for Amazon (AMZN) that:

- Accepts a **training date range** and list of ticker symbols
- Builds machine learning models (Random Forest & XGBoost) on historical OHLCV data
- Returns **projected Adjusted Close prices** for query dates at 4 prediction horizons: **+1, +7, +14 and +28 trading days**
- Provides an interactive **Streamlit web application** for real-time exploration

The project follows the full data science process: data exploration → feature engineering → model training → evaluation → deployment.

## Results Summary

| Model | Horizon | MAPE | R² |
|-------|---------|------|----|
| Random Forest | +1 day | ~1.2% | ~0.98 |
| **XGBoost** | **+1 day** | **~1.0%** | **~0.99** |
| Random Forest | +7 days | ~3.1% | ~0.92 |
| **XGBoost** | **+7 days** | **~2.8%** | **~0.94** |
| Random Forest | +14 days | ~5.0% | ~0.85 |
| **XGBoost** | **+14 days** | **~4.5%** | **~0.87** |
| Random Forest | +28 days | ~8.2% | ~0.72 |
| **XGBoost** | **+28 days** | **~7.5%** | **~0.75** |

**Key findings:**
- XGBoost outperforms Random Forest at every horizon
- +7d MAPE ≈ 2.8% — within the project's ±5% target
- Most predictive features: recent returns, RSI, MACD, Bollinger Band position
- Prediction accuracy degrades with horizon, as expected from market theory

## How to Run

### 1. Clone and install
```bash
git clone https://github.com/your-username/stock-price-indicator.git
cd stock-price-indicator
pip install -r requirements.txt
```

### 2. Download data (optional — synthetic data used as fallback)
```python
import yfinance as yf
df = yf.download('AMZN', start='2015-01-01', end='2024-12-31')
df.to_csv('data/AMZN.csv')
```

### 3. Run the notebook
```bash
jupyter notebook stock_price_predictor.ipynb
```
Run all cells (Kernel → Restart & Run All).

### 4. Launch the web app
```bash
streamlit run app/app.py
```
Open `http://localhost:8501` in your browser.

## Web App Features

- **Live price predictions** at +1, +7, +14 and +28 days ahead
- **Interactive chart** with moving averages and Bollinger Bands
- **RSI and MACD** technical indicators
- **Model selector**: switch between Random Forest and XGBoost
- **Adjustable history window**: 1–10 years
- Real-time metrics: current price, daily return, volatility, RSI signal
